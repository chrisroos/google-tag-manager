Testing Google Tag Manager

This example demonstrates how to use the GTM Data Layer to send an Enhanced Ecommerce checkout event when a product link is clicked.

Decorate the product link with data attributes containing information required by the ecommerce checkout event.

Use jQuery to attach a function to the click event of the link. This function extracts the information from the data-* attributes, uses it to construct a GTM ecommerce checkout event and adds the event to the data layer.

There are no guarantees that the event will be sent to GTM before the browser navigates to the target of the link.
